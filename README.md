# My Personal Website

URL: https://finnhb.github.io/

My personal website and blog, built with [Hugo](https://gohugo.io/) and the
[PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme. Blog posts are
authored with R [blogdown](https://bookdown.org/yihui/blogdown/) — each post is an
`index.Rmd` that is knitted to `index.html`. The site is deployed to GitHub Pages
via GitHub Actions on every push to `main`.
